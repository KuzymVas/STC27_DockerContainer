How To:

To access environment variables via REST service:

  • Go to /env to see the list of all environment variables
  • /env?name=JAVA_HOME allows you to query a value of a single environment variable JAVA_HOME
  • /env/JAVA_HOME is another way to access value of the environment variable JAVA_HOME
